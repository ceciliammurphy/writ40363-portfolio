## üé® Design System Reference

### CSS Variables
All colors, spacing, and typography are defined as CSS custom properties in `:root`:

```css
--color-black
--color-white
--color-red
--color-silver
--space-xs, --space-sm, --space-md, --space-lg, --space-xl, --space-2xl
--font-primary (Montserrat)
--font-display (Playfair Display)
```

### Key Components
- `.hero` - Hero/header section
- `.section-header` - Centered section titles with dividers
- `.skill-card` - Individual skill cards with hover effects
- `.project-card` - Project showcase cards
- `.btn-primary` / `.btn-secondary` - CTA buttons
- `.contact-link` - Contact item with icon
